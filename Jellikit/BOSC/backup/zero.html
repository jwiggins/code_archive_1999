Zero Updated (Again!)

27 February 1998 (YABOSCU) - Full MIP-mapping and page flipping support have been added to the BeOS version.

Zero Updated

24 February 1998 (YABOSCU) - Zero, the 3D game for BeOS, is progressing smoothly. Developers implemented code to allow cool gradient effects for sky and horizon rendering. More improvements were made to the Dirt Mound terrain editor. Also, vast improvements were made to the automatic landscape texturer, which will save a lot of time during level creation. Developers also made the illumination code more intelligent.

Zero Updated

15 February 1998 (YABOSCU) - Here is an early screen shot of Dirt Mound, the Zero terrain editor. The texture placement controls still need work, but overall the app is quite nice to use. By the way, this will be released with Zero.

Zero is a 3-D action and adventure game with a science fiction theme that will be released for MacOS as well as BeOS.
