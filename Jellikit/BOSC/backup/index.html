Top News [April 2]

Got News? mail 40@hlserver.com.

[Run v2.1]
Run v2.1 was posted on Be's FTP servers today. Intel and PowerPC binaries are available at: http://www.be.com/beware/Utilities/Run.html Run is a BeOS utility based on the Run command in the Win95 start menu. It allows you to run shell commands without opening a Terminal. A Tracker add-on is also included. The new version makes some cosmetic improvements, adds resizable windows, and is compatible with R3/Intel. Matt Brubeck.

[BeCheckPoint]
John Wiggins has released BeCheckPoint its a login type application. When launched, it kills the Tracker and deskbar, (and mail_daemon if its running). After killing off all the useful stuff :) you get a password prompt. If you enter the incorrect password or you quit before entering a password, you get rebooted, otherwise everything returns to normal and the app exits. A future release will mask the password as you enter it.

[Epson Stylus]
EPSON Stylus Printer Driver for BeOS PR2 This driver actually supports the following EPSON printers: - EPSON Stylus Color - EPSON Stylus Pro - EPSON Stylus Pro XL - EPSON Stylus Pro XL+. Download.

[PostgresSQL]
It's an early version of a postgres port to BeOS. Postgres is a full featured Relational Database system (see www.postgres.org). It is composed by a "backend" which is the database system itself and a client library on which it's possible to build complete client applications. Download.

[SheepShaver]
The SheepShaver public beta is now available on http://www.sheepshaver.com/ and BeWare (it may take a few days to show up on ftp.be.com)! This beta will expire on April 20 and requires BeOS/PowerPC R3. Preview Release 2 will not work. Unless you are betatesting R3 for Be, you can't run SheepShaver for now because R3 is not going to be released before mid-April. The beta is not "crippled" in any way, but there are some features like sound input that they couldn't get done in time.
