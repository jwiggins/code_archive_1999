[Run]
This is a small app which allows you to enter and run a shell command from a small window. It's based on the "Run" command in the Win95 start menu. Also includes a Tracker add-on which uses the current folder as the working directory (very useful). In version 2.1, the window is now resizeable, and various cosmetic improvements have been made. Download, Source code is available here.
