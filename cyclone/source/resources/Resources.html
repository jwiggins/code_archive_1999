Application

The Cyclone Application is tenatively of MIME type "application/x-be-executable", with Application Signature "application/x-vnd.cyclone-app" and should support the add-ons as documents which it could open. The Name of this type should be "Cyclone Application." Double clicking on an add-on should load the application normally but if it is an interface add-on, it should use that as the interface, and if it is a command add-on, it should load that add-on first. Similar rules should apply for multiple refs.

Command Add-Ons

Cyclone Command Add-Ons should not be of type "application/x-be-executable" like most add-ons, they should instead be of MIME type "applicatin/x-vnd.cyclone-command" with a name of "Cyclone Command Add-On." The default application for the document type should be "Cyclone." 

$ listattr standard
file standard
  Type         Size                 Name
----------  ---------   -------------------------------
0x52415754         20                        _trk/pinfo
  MIME str         34                         BEOS:TYPE

Interface Add-Ons

Cyclone Interface Add-Ons should not be of type "application/x-be-executable" like most add-ons, they should instead be of MIME type "applicatin/x-vnd.cyclone-interface" with a name of "Cyclone Interface Add-On." The default application for the document type should be "Cyclone."
